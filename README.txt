LOGICSOFT PWA

Archivos incluidos:
- index.html
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png
- icons/apple-touch-icon.png

Para instalar:
1. Sube toda esta carpeta a tu hosting.
2. Abre index.html desde HTTPS o localhost.
3. La app podrá instalarse desde el navegador.

Si ya tenías una versión anterior en caché, haz una recarga forzada o borra el service worker viejo.
